# Fix-Rent

## Descripción del proyecto
Fix & Rent es una app que permite solicitar reparaciones, hacer seguimiento en tiempo  real, pagar en línea y, además, gestionar la renta de los inmuebles. El usuario puede  contratar técnicos certificados, recibir reportes y garantizar el mantenimiento continuo de su propiedad. 

> **Propuesta de valor:**
*Transforma, publica y arrienda fácil."*
